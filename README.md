# ⚡ AxiomEngine
## 🧠 Computer Algebra System (CAS) in Python

A lightweight mathematical engine for symbolic computation, calculus, equation solving, and graph visualization.

---

# 🚀 Overview

AxiomEngine is a Python-based math computation system designed to perform:

- Algebraic simplification  
- Equation solving  
- Differentiation & integration  
- Function plotting  
- System of equations solving  

It works like a **mini Wolfram Alpha engine (lite version)**.

---

# ✨ Features

## 🧮 Mathematical Engine
- Expression evaluation  
- Symbolic simplification  
- Algebraic transformations  

## 📐 Calculus Module
- Derivatives (∂)  
- Integrals (∫)  

## 🔢 Equation Solver
- Single variable equations  
- System of linear equations  

## 📊 Visualization
- Function graph plotting using Matplotlib  

## 📜 Utility
- Calculation history tracking  

---

# 🛠 Tech Stack

## Core Technologies
- Python 🐍  

## Libraries Used
