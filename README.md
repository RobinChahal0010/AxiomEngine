# 🧠 About AxiomEngine

AxiomEngine is a lightweight **Computer Algebra System (CAS)** built using Python that performs symbolic mathematics, calculus operations, equation solving, and graph visualization.

It is designed as a mini mathematical engine inspired by systems like **Wolfram Alpha**, focused on understanding how symbolic computation systems work internally.

---

# ⚡ Engine Capabilities

## 🧮 Symbolic Computation

- Algebraic simplification
- Expression evaluation
- Symbolic transformations
- Mathematical parsing

---

## 📐 Calculus Engine

- Differentiation
- Indefinite Integration
- Symbolic calculus operations

### Example

```python
diff(x**3 + 2*x, x)
```

### Output

```python
3*x**2 + 2
```

---

## 🔢 Equation Solver

### Single Variable Equations

```python
solve(x**2 - 4, x)
```

### Output

```python
[-2, 2]
```

---

## 🧩 Linear System Solver

```python
x + y = 5
x - y = 1
```

### Output

```python
x = 3
y = 2
```

---

## 📊 Visualization Engine

Plot mathematical functions dynamically using Matplotlib.

### Supported Graph Types

- Polynomial Functions
- Trigonometric Functions
- Exponential Functions
- Custom Expressions

---

# 🏗 Internal Architecture

```text
User Input
    ↓
Expression Parser
    ↓
Symbolic Engine
    ↓
Transformation Layer
    ↓
Solver / Calculus Module
    ↓
Graph Rendering Engine
    ↓
Final Output
```
