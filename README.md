🚀 ⚡ AxiomEngine










🧠 A powerful Computer Algebra System (CAS) built in Python for symbolic mathematics, calculus, equation solving, and graph visualization — inspired by Wolfram Alpha (Lite Engine).

🔥 What is AxiomEngine?

AxiomEngine is not just a calculator.

It is a mini mathematical computation engine that can:

Solve equations like a symbolic solver
Perform calculus operations (derivatives & integrals)
Simplify complex algebraic expressions
Plot mathematical graphs dynamically
Store computation history like a smart system
⚡ Core Features

🧮 Scientific Expression Evaluation
📐 Symbolic Algebra Simplification
📊 Equation & System Solver
📈 Graph Plotting Engine (Matplotlib)
∂ Derivatives (Symbolic Differentiation)
∫ Integrals (Symbolic Integration)
🧠 Smart Calculation History

🚀 Installation & Run

cd AxiomEngine
pip install -r requirements.txt
python main.py
💡 Example Usage
➤ Algebra
Input: (x + 1)^2
Output: x^2 + 2x + 1
➤ Derivative
Input: x^3 + 2x
Output: 3x^2 + 2
➤ Integral
Input: x^2
Output: x^3 / 3
